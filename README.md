# Al-ML-Internship-Day6
🧠 1. Steps in Problem Solving
Understand the problem

👉 Know input, output, conditions
Break into smaller steps
👉 Divide complex problem into simple parts
Write logic (pseudocode)
👉 Plan solution in simple English
Convert to code
👉 Write program (Python)
Test and debug
👉 Check with different inputs and fix errors
📌 2. What is Algorithm?

👉 An algorithm is a step-by-step procedure to solve a problem.

Example:

Find sum of numbers:

Take input
Add numbers
Print result

👉 Simple + clear steps = Algorithm

🔄 3. What is Menu-Driven Program?

👉 A program that shows multiple options (menu) and performs actions based on user choice.

Example:
1. Add
2. Subtract
3. Exit

👉 User selects option → program runs that task

🤖 4. Why Logic is Important in AI?
AI works on data + logic
Helps in:
Data filtering
Pattern recognition
Decision making
Without logic → no correct output

👉 Logic = Brain of AI
